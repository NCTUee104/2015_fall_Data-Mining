# Data Mining

### Prerequisite :
* Data structure
* DataBase

### Textbooks :
*  Introduction to Data Mining, Pang-Ning Tan, Michael Steinbach and Vipin Kumar, Addison-Wesley

### Grading Policy : 
* one midterm exam
* participate competitions in Kaggle

### Course descriptions :
本課程將介紹資料探勘的基本理論與相關的技術, 並透過專案實作的方式, 使學生能將所學習之資料探勘技術實際用在專案中。
